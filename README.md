## 前端快速开发模版

### 工具链
* 构建工具gulp
* 编译打包工具browserify
* 代码编译babel
* css兼容autoprefixer
* css预编译less
* html代码预编译模版swig
* 前端代码版本更新rev

### 公共类库
* jQuery
* Zepto
* 自己开发的简洁单页面框架app.js
* 图标fontello
* 字体reboto
* 基础样式base.css

### 文件目录结构
* css: css文件
* less: less文件
* js: js文件
* dist: 打包生成的项目文件
* layout: html母板
* lib: 公共类库
* template: html页面

### 使用
```bash
npm install
npm run build
```
